Operators, Objects, and Functions

Operators

  • + - * / % ++ += -= > < <= >= !
  • = == ===
    = is used for assigning values to a variable in JavaScript.
    == is used for comparison between two variables irrespective of the datatype of variable.
    === is used for comparision between two variables but this will check strict type, which means it will check datatype and compare two values.
  • & && | ||
  • [] : .
  • typeof

Objects

Objects are user defined datatypes with properties. like key-value.

Functions

Anonymous Functions

  • Functions with no name.

Function Expression

  • Anonymous function assigned to a variable.

IIFE: Immediately Invoked Function Expressions.

  • No explicit calling is required.
  • One time usage, right when the function is defined it gets invoked.
  • To perform a task for event handlers, listeners, and callbacks — anonymous functions
  • To maintain data privacy
  • To use as an argument when a function is called — anonymous functions & IIFE
  • Prevent conflicts between script files that could have the same variable names — anonymous functions & IIFE

Arrow Functions

  • Arrow functions are from ES6 standards
  • An arrow function is a syntactical change of a js function.
  • They help us write clean and concised code.
  • var foo = (p1, p2) => {statements}
  • this Object and Arrow Functions

Object Construct Functions

  • Were used prior to ES6. Later ES6 introduced classes to simplify this.
  • We follow a notation of first letter as capatal called Pascal notation - just to differentiate from a regular function.
